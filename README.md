# docs-portfolio

This repository contains a Zensical documentation site.

## Quick start

1. Install Zensical: `pip install zensical`
2. Build the site: `zensical build --clean`
3. Preview locally: `zensical serve`

## Notes

- Source content is in `docs/`
- Generated site output is in `site/`
- GitHub Pages deployment is configured in `.github/workflows/docs.yml`
